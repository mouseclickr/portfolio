I

Existing

This project has a real website as starting point, built with a template that isn't always doing what it should. Since I like its design, I didn't want to do a complete makeover, but try to improve its drawbacks:

  1. the font is too small for large screens
  2. the top right buttons are small and change on hover is barely visible
  3. home page sliding Carousel is a bit lagging on large screens
  4. "Work" page has an uneven grid, hardly responsive - some images appear smaller than the rest on mobile
  5. image load is slow
  6. Lightbox is slow and changes size as images change size
  7. "About" page has very long lines of text, that makes reading annoying
  8. the footer isn't fixed to bottom, which on large screens is visually distracting
  9. Facebook icon isn't visible on desktop using Firefox, but is visible in Chrome and it makes the vertical scrollbar visible on the home page
  10. current year doesn't update (see Copyright line in footer)
Proposal

Small changes make for an improved user experience:

  1. changed the fonts for nav bar and other texts, to be scalable according to screen size
  2. buttons are bigger and change on hover is substantial
  3. changed Carousel from sliding to fade-in
  4. the image grid is even, with multiple breakpoints for every screen size
  5. image load is much faster, and one can no longer see a lag
  6. improved Lightbox, that stays in place, no matter the image size
  7. redid the entire "About" page, with scalable fonts and lines no longer than 80ch
  8. the footer is fixed to bottom on pages where it should (for medium and large screens) and has a different layout
  9. all icons are visible on both mobile and other screens - I also included location and links to mail and phone apps, making the contact page redundant
  10. current year updates automatically, with a minimal script.js
