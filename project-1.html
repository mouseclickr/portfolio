WEBSITE FOR A THERAPIST

SEE THIS PROJECT

I

Existing

Doing research for this type of website (see here, here, here, and here), I noticed clear areas of improvement:

  1. content is poorly structured - hard to navigate and/or lacking basic information
  2. text is either too small or too hard to read over the background image
  3. site is not built with a "mobile first" approach
  4. some images are unrelated to content
  5. one of the websites in question has a Shop page for different porcelain sets - why?
  6. site isn't loading fast enough

Proposal

Putting myself in the shoes of people in need of therapy, I wondered what type of questions would I have looking for a mental health professional. In doing so, I addressed the issues I found in my research:

  1. I want to create a website that gives people a firm knowledge of what to expect before contacting a therapist
  2. text should be at least AA rated with fluid typography, that scales according to screen size
  3. responsive design is a MUST
  4. convey messages effectively through imagery
  5. if you have other areas of interest, do a separate site for that and if you must, link it from this one
  6. fast loading sites can be done using math formulas and grid/flexbox layouts, there's no need for hard coded scripts
